QWERTY Keyboard for MRI

Request Quote

Overview

A USB QWERTY keyboard with 104 standard keys that works inside an fMRI machine.

Features

  • 104 standard keys are easy for subjects to use with no adjustment from a typical keyboard
  • Standard USB keyboard interface – works with all modern operating systems (Windows, Mac, Linux)
  • Rubber membrane design
  • Shipping weight: 8 pounds
  • Dimensions: 15 inches length, 6 inches deep, 2 inches thick
  • Pair with the small touchpad with the custom controller for a normal computer interaction experience
  • For research use only
